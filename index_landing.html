Hoodie enables you to express yourself through technology by making web application development very fast, easy and accessible.

It’s a a complete backend for your apps, ready for you to get creative: develop your frontend code, plug it into our frontend-friendly API and your app is ready.

Hoodie is Open Source Software.

Features

Hoodie is an Offline First, noBackend architecture. Its Dreamcode API gives you user signup and administration, data storage, loading, synchronisation and shares, emails and payments and can be extended with plugins. Hoodie is written in JavaScript and Node.JS and relies on CouchDB.
